# LetsMemocard
Juego de memoria creado para jugar con amigos, familiares y en solitario! no hay solo memoria hay cuentas con contraseña y logros, niveles y mas!  puedes guardarlas con un boton de importar/exportar con un archivo .json que puedes descargar para poner en una nueva computadora y que tus cuentas no desaparezcan!
